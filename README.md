# project-git

Простой Windows Forms проект на C#:
- При нажатии на кнопку открывается новое окно с текстом.
- Можно ввести текст в поле и показать его в новом окне.
- Демонстрация работы с ветками, Pull Request и GitHub Actions.

## Как запустить
1. Открыть решение в Visual Studio.
2. Запустить проект (F5).

## Ветки
- main — основная
- feature/new-style — изменения UI
